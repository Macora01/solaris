# 🌌 Sistema Solar Interactivo 3D

Simulación interactiva del sistema solar con información detallada de planetas y lunas.

## 🚀 Características

- Vista cenital y 3D
- Información detallada de cada cuerpo celeste
- Órbitas animadas
- 10+ lunas principales
- Cinturón de asteroides

##  Controles

- **Click** en planetas/lunas para ver información
- **Arrastrar** para mover la vista
- **Rueda** del mouse para zoom
- **Espacio** para pausar/reanudar
